  2. p : used for obtaining distribution function

  3. q : used for obtaining quantile function

Examples

Discrete distribution

  1. If x ~ B(8,0.5), find p(x=3) & F(2)

solution:

R code to find p(x=3) and F(2)

rp1 = dbinom(3,8,0.5)

rp2 = pbinom(2,8,0.5)

  1. Draw a r.s. of size 8 from binomial distribution with parameters n=10 & p= 0.4

solution:

R code to obtain random sample

rs=rbinom(8,10,0.4)

Continuous distribution

  1. If x ~ N(5,4), find p(x<2) & p(12<x<15)

solution:

R code to find p(x<2) and p(12<x<15)

rp3 = pnorm(2,5,2)

rp4 = pnorm(15,5,2)-pnorm(12,5,2)

  1. Draw a r.s. of size 20 from normal distribution with mean=1 & var=4

solution:

R code to obtain random sample

rs=rnorm(20,1,2)

Density plot in R

1) PLot density function in R

2) Density curve with histogram

3) Multiple density curves in R

4) How to fill area under density curve

5) Density plot using ggplot2

1)PLot density function in R

set.seed(1234)
x=rnorm(2000)
plot(density(x),xlab = "x-values",ylab ="Density values",col="red")

2) Density curve with histogram

set.seed(1234)
x=rnorm(2000)
hist(x,freq = FALSE, main="Histogram and Density")
dx=density(x)
lines(dx,lwd=2,col="red")

3) Multiple density curves in R

set.seed(1234)
x=rnorm(2000)
dx=density(x)
plot(dx,lwd=2,col="red",main="Multiple curves")
y=rnorm(2000)+1;
dy=density(y)
lines(dy,col="blue",lwd=2)

4) How to fill area under density curve

plot(dx, lwd = 2, col = "red", xlim = c(-4, 6), ylim = c(0, 0.5),main="Shaded Density Curve")
polygon(dx, col = "red")

5) Density plot using ggplot2

library(ggplot2)

df <- data.frame(x = x)

ggplot(df, aes(x = x)) + 
       geom_density(color = "red",fill = "red", alpha = 0.6)
